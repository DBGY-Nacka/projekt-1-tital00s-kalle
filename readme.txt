Product Hub Webiste

This is a webiste for a product shopping company. The purpose is to create a usable website 
and explore how a website can be made more trustworthy. It solves the issue with many websites feeling insecure 
for the customer. 

It is developed using html, javascript and css to style the webiste. Combined with python and flask.

Libraries needed to download:
flask
sqlite3

To run the code, simply run app.py. Then follow the given local host link to view the website.
